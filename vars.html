Les types de déclarations

//!\\ Les types de déclarations sont différents des types de données (cf partie suivante) !

Il existe trois types de déclarations de variable en JavaScript :

  • let : la plus utilisée
  • const : la valeur ne pourra pas être modifié
  • var : très proche de let, et de moins en moins utilisé

Exemples :


        let myVar; // Initialiser la variable sans lui attribuer de valeur
        let myVar2 = "ma première variable"; // Initialiser la variable en lui attribuant une valeur
        myVar = "ma seconde variable"; // Modification de la variable myVar. Sa valeur n'est plus null.
        myVar2 = "je modifie ma première variable"; // Modification de la variable myVar2
        let myNumber = 2.5;
        let myBoolean = true;
        
        const myConst = "ma première constante"; // Initialiser la constante en lui attribuant une valeur (c'est obligatoire pour une constante)
        myConst = "cela va générer une erreur"; // Il n'est pas possible de modifier une constante. Le script s'arrêtera ici et affichera une erreur dans la console.
        const myNumberConst = 3.4;
        const myBoolean = false;

Les types de variables

Le dernier standard ECMAScript définit 8 types de données :

  • Chaîne de caractères (String)
  • Nombre
  • Booléen
  • Null
  • Undefined
  • BigInt (proposition pour ES2020)
  • Symbole (type introduit avec ECMAScript 6)
  • Objet

Exemples :


        let myString = "ceci est une chaîne de caractère";
        let myString2 = "12";
        let myNumber = 15;
        let myNumber2 = 15.34;
        let myBoolean = true;
        let myBoolean2 = false;
        let myNull = null;
        let myUndefined = undefined;
        
        console.log(typeof myString); // Permet de vérifier le type de donnée

Les objets

Un objet peut être vu comme un ensemble de variables. Chacune d'elle étant accessible via une "clé". Une clé peut être une chaîne de caractères ou un nombre.

Exemples :


        let myObject = {
            maCle1: "ma valeur 1",
            maCle2: "ma valeur 2",
            maCle3: "ma valeur 3",
        }; // Initialisation de l'objet, comprenant 3 variables, accessibles via une clé en chaîne de caractère. Dans ce cas, on peut dire que ces variables sont des attributs de l'objet
        myObject.maCle1 = "ma nouvelle valeur 1"; // Modification de la première variable (propriété)
        console.log(myObject.maCle1); // Affiche : "ma nouvele valeur 1". On accède à la valeur grâce à la clé associée (au nom de la propriété)
        
        let myArray = [
          "ma valeur 1",
          "ma valeur 2",
          "ma valeur 3",
        ]; // Initialisation d'un tableau, comprenant 3 variables, accessibles via une clé en nombre
        myArray[0] = "ma nouvelle valeur 1"; // Modification de la première variable
        console.log(myArray[0]); // Affiche : "ma nouvele valeur 1". On accède à la valeur grâce à la clé associée

Récapitulatif

Une variable a donc : un type de déclaration, un type de donnée, et une valeur.

Exemples :


        const firstname = "Jean";
        let age = 20;
        const isMajor = age >= 18;
        let friends = ["Jacques", "Pierre", "Paul"];
        let work = {
          company: "OpenClassrooms",
          job: "student",
          from: "2021-01-01",
        };
        
        // Qu'afficheront ces logs ?
        console.log(firstname);
        console.log(typeof firstname);
        console.log(typeof age);
        console.log(isMajor);
        console.log(friends[1]);
        console.log(friends.length);
        console.log(typeof work);
        console.log(work.job);
