Exercices sur les flexbox

Introduction

Pour chaque exercice, ajouter le code manquant.

Voici un exemple de consigne :

Code initial :

See the Pen Untitled by Lola (@Milalea) on CodePen. 


  console.log('Jean a 10ans');
  console.log('Jacques a 10ans');
  console.log('Pierre a 10ans');

Voici un exemple de solution attendue :


  function display(name) {
    console.log(name+' a 10ans');
  }
  display('Jean');
  display('Jacques');
  display('Pierre');

Conseils

Etape 1 :

Bien observer les éléments communs à chaque ligne (en vert ci-dessous), et ceux qui diffèrent (en rouge).

console.log('Jean a 10ans');
console.log('Jacques a 10ans');
console.log('Pierre a 10ans');

Etape 2 :

Créer une fonction, et y mettre le code commun


  function display() {
    console.log(' a 10ans');
  }

Etape 3 :

Remplacer le code qui diffère, par un paramètre


  function display(name) {
    console.log(name+' a 10ans');
  }

Etape 4 :

Appeler la fonction pour qu'elle soit exécutée, avec les bons paramètres


  function display(name) {
    console.log(name+' a 10ans');
  }
  display('Jean');
  display('Jacques');
  display('Pierre');

Il faudra appliquer cette même méthodologie sur chacun des exercices suivants.

Exercices

Le code html de base sera toujours le suivant :


  <div id="element1">Element 1</div>
  <div id="element2">Element 2</div>
  <div id="element3">Element 3</div>

Exercice 1

Code à améliorer :


  document.getElementById('element1').innerHTML = "1er changement de texte";
  document.getElementById('element1').innerHTML = "2ème changement de texte";
  document.getElementById('element1').innerHTML = "3ème changement de texte";

Exercice 2

Code à améliorer :


  document.getElementById('element1').style.color = "red";

  document.getElementById('element2').style.color = "red";

  document.getElementById('element3').style.color = "red";

Exercice 3

Code à améliorer :


  document.getElementById('element1').innerHTML = "Lorem";

  document.getElementById('element3').innerHTML = "dolor";
