Objets et Fonctions

Exercice 1


  // Soit l'objet suivant :
  const Person = {
    firstname: "Jean",
    age: 20,
    friends: ["Jacques", "Pierre", "Paul"]
    currentWork: {
      company: "OpenClassrooms",
      job: "Étudiant",
      from: "2021-01-01",
    },
    previousWorks: [
      {
        company: "EDF",
        job: "Développeur",
        from: "2020-06-01",
        to: "2021-01-01",
      },
      {
        company: "SFR",
        job: "Stagiaire",
        from: "2020-01-01",
        to: "2020-06-01",
      },
    ]
  };

Pour chacune des lignes ci-dessous, créer une fonction "getter" qui retournera l'information demandée, et/ou une fonction "setter" qui la modifiera (voir l'exemple attendu ci-après).

  • L'âge (getter + setter)
  • Le nombre d'amis (getter)
  • Le nom de l'entreprise actuelle (getter + setter)
  • Le nom du poste occupé lorsqu'il travaillait chez "SFR" (ou autre nom d'entreprise passée en paramètre) (getter + setter)
  • Supprimer l'ami "Pierre" (ou autre nom d'ami passé en paramètre) (setter)
  • Ajouter une précédente expérience professionnelle (setter)
  • Indiquer un nouveau travail actuel (avant cela, passer l'expérience pro actuelle dans les précédentes expériences pro) (setter)

Exemple


  // - Firstname (getter + setter)
  function getFirstname(person) { // <= Getter
    return person.firstname;
  }
  function setFirstname(person, newName) { // <= Setter
    person.firstname = newName;
    return person;
  }
