Exercices sur les fonctions

Pour chaque exercice, il faudra créer une fonction pour éviter les duplications.

Aucune recherche Google ou autre n'est nécessaire. Son utilisation est donc innterdite.

Le code html de base sera toujours le suivant :


  <div id="element1">Element 1</div>
  <div id="element2">Element 2</div>
  <div id="element3">Element 3</div>

Voici un exemple de consigne :

Code à améliorer :


        console.log('Jean a 10ans');
        console.log('Jacques a 10ans');
        console.log('Pierre a 10ans');

Voici un exemple de solution attendue :


        function display(name) {
          console.log(name+' a 10ans');
        }
        display('Jean');
        display('Jacques');
        display('Pierre');
