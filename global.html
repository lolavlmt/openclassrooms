Pour chacune des lignes ci-dessous, créer une fonction "getter" qui retournera l'information demandée, et une fonction "setter" qui la modifiera (cf exemple attendu ci-après).

  • L'âge
  • Le nombre d'amis
  • Le nom de l'entreprise actuelle
  • Le nom du poste occupé lorsqu'il travaillait chez "SFR" (ou autre entreprise passée en paramètre)

Exemple


  // - Firstname 
  function getFirstname(person) { // <= Getter
    return person.firstname;
  }
  function setFirstname(person, newName) { // <= Setter
    person.firstname = newName;
    return person;
  }
