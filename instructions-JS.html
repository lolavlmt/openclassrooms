Décomposer et comprendre une instruction JS

Introduction

Le javascript est un langage, tout comme le français. On peut donc essayer de faire certains parallèles.
Voici deux phrases en Français : "Le chien(1) est de taille(2) moyenne(3). Il mange(4) de la nourriture(5)".
Et comment on pourrai les décomposer :

  1. Sujet
  2. Caractéristique
  3. Adjectif
  4. Verbe d'action
  5. Complément d'objet

La traduction de ces phrases en JS serai :


theDog.size = "medium";
theDog.eat("food");

Et la décomposition se nommerai ainsi :

  1. Variable
  2. Attribut
  3. Valeur
  4. Fonction
  5. Paramètre

Il arrive qu'en JS, la fonction représente à la fois le verbe d'action + le complément d'objet.
Par exemple : Il(1) mange de la nourriture(2) froide(3).
En JS :


        theDog.eatFood("cold");

Ici, le chien mange toujours de la nourriture, ce n'est pas utile de passe "food" en paramètre.

Ces phrases sont affirmatives. La traduction en JS que l'on vient de faire, correspond au mode "écriture" (on modifie la valeur de la propriété).
Lorsque la phrase est à l'interrogative, on passe en mode "lecture".
Par exemple : Quelle est la taille du chien ?
Traduction JS :


  console.log(dog.size);
  // ou bien
  let size = dog.size; // Pour les exercices suivant, on utilisera cette écriture plutôt que celle du console.log ci-dessus.

Une autre équivalence que l'on peut trouver entre le français et le JS, est celle du "complément du nom".
Par exemple : Le chien(1) de mon ami(2) est de taille moyenne.

  1. Sujet
  2. Complément du nom

Traduction JS :


  myfriend.dog.size = "medium";
  1. Variable
  2. Propriété

Exercices

Dans les prochains exercices, il y aura 2 phrases en français, et 1 instruction JS.
Il faudra annoter les 2 phrases données a) et b) avec des numéros (1, 2, ... comme dans les exemples ci-dessus), et indiquer à quoi correspondent les numéros (Sujet, Verbe, etc).
Puis, dans un second temps, il faudra traduire ces 2 phrases distinctes en code JS. On ne cherche pas ici à produire du code "réaliste", on cherche à bien identifier les éléments qui construisent une instruction, et comment on les assemble entre eux.
À partir de l'instruction JS c), il sera demandé de faire l'exercice en sens inverse : Il faudra l'annoter et donner les termes JS correspondant (Variable, Attribut, etc). Pour finir, il faudra traduire cette instruction en français.

Exemple :

a) La maison est de forme carrée.
b) ... // Pas besoin pour l'exemple
c) footer.opacity = 1;

Solution attendue :

a) La maison(1) est de forme(2) carrée(3).
1. Sujet
2. Caractéristique
3. Adjectif
Traduction JS : house.forme = "square";
b) ...
c) footer(1).opacity(2) = 1(3);
1. Variable
2. Attribut
3. Valeur
Traduction Fr : Le pied de page a une opacité de 1;
