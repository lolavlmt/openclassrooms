Introduction

Voici un rappel des opérateurs de comparaison et des opérateurs logique.

Opérateurs de comparaison

Opérateur Description

Soit la variable : let myVar = 12;
== pour tester l'égalité. Il faut doubler le = pour le différencier de la déclaration des variables. 

  let result1 = myVar == 12;
  let result2 = myVar == 35;
  let result3 = myVar == 8;
  console.log(result1, result2, result3); // true  false  false
!= différent de 

  let result1 = myVar != 12;
  let result2 = myVar != 35;
  let result3 = myVar != 8;
  console.log(result1, result2, result3); // false  true  true
< strictement inférieur 

  let result1 = myVar < 12;
  let result2 = myVar < 35;
  let result3 = myVar < 8;
  console.log(result1, result2, result3); // false  true  false
<= inférieur ou égal 

  let result1 = myVar <= 12;
  let result2 = myVar <= 35;
  let result3 = myVar <= 8;
  console.log(result1, result2, result3); // true  true  false
> strictement supérieur 

  let result1 = myVar > 12;
  let result2 = myVar > 35;
  let result3 = myVar > 8;
  console.log(result1, result2, result3); // false  false  true
>= supérieur ou égal 

  let result1 = myVar >= 12;
  let result2 = myVar >= 35;
  let result3 = myVar >= 8;
  console.log(result1, result2, result3); // true  false  true

Opérateurs logiques

Opérateur Description

Soit les variables : let varTrue = true; let varFalse = false;
&& ET. Les 2 valeurs doivent être à "true", pour que le résultat soit "true". 

  let result1 = varTrue && varFalse;
  let result2 = varTrue && true;
  let result3 = false && varTrue;
  console.log(result1, result2, result3); // false  true  false
|| OU. Au moins une des 2 valeurs doit être égale à "true", pour que le résultat soit "true". 

  let result1 = varTrue || varFalse;
  let result2 = varFalse || false;
  let result3 = varFalse || true;
  console.log(result1, result2, result3); // true  false  true
! NON. Inverse le résultat. 

  let result1 = !varTrue;
  let result2 = !varFalse;
  let result3 = !(varTrue && varFalse);
  console.log(result1, result2, result3); // false  true  true
