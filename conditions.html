Exercices sur les conditions

Introduction

Voici un rappel des opérateurs de comparaison et des opérateurs logique.

Opérateurs de comparaison

Opérateur Description

Exemple avec la variable : let myVar = 12;
== pour tester l'égalité. Il faut doubler le = pour le différencier de la déclaration des variables. 

  let result1 = myVar == 12;
  let result2 = myVar == 35;
  let result3 = myVar == 8;
  console.log(result1, result2, result3); // true  false  false
!= différent de 

  let result1 = myVar != 12;
  let result2 = myVar != 35;
  let result3 = myVar != 8;
  console.log(result1, result2, result3); // false  true  true
< strictement inférieur 

  let result1 = myVar < 12;
  let result2 = myVar < 35;
  let result3 = myVar < 8;
  console.log(result1, result2, result3); // false  true  false
<= inférieur ou égal 

  let result1 = myVar <= 12;
  let result2 = myVar <= 35;
  let result3 = myVar <= 8;
  console.log(result1, result2, result3); // true  true  false
> strictement supérieur 

  let result1 = myVar > 12;
  let result2 = myVar > 35;
  let result3 = myVar > 8;
  console.log(result1, result2, result3); // false  false  true
>= supérieur ou égal 

  let result1 = myVar >= 12;
  let result2 = myVar >= 35;
  let result3 = myVar >= 8;
  console.log(result1, result2, result3); // true  false  true

Opérateurs logiques

Opérateur Description

Exemple avec les variables : let varTrue = true; let varFalse = false;
&& ET. Les 2 valeurs doivent être à "true", pour que le résultat soit "true". 

  let result1 = varTrue && varFalse;
  let result2 = varTrue && true;
  let result3 = false && varTrue;
  console.log(result1, result2, result3); // false  true  false
|| OU. Au moins une des 2 valeurs doit être égale à "true", pour que le résultat soit "true". 

  let result1 = varTrue || varFalse;
  let result2 = varFalse || false;
  let result3 = varFalse || true;
  console.log(result1, result2, result3); // true  false  true
! NON. Inverse le résultat. 

  let result1 = !varTrue;
  let result2 = !varFalse;
  let result3 = !(varTrue && varFalse);
  console.log(result1, result2, result3); // false  true  true

Les amis de mes amis sont mes amis : true && true = true.
Les amis de mes ennemis sont mes ennemis : true && false = false.
Les ennemis de mes amis sont mes ennemis : false && true = false.
Les ennemis de mes ennemis sont mes ennemis : false && false = false.

Les règles de priorité

Lorsqu'il y a plusieurs opérateurs logiques, il faut les prendre dans l'ordre, sauf s'il y a des parenthèses.

Exemples :


let result = varFalse && varFalse || varTrue;
// Décomposons, dans l'ordre :
let subResult1 = varTrue && varFalse; // false
let subResult2 = subResult1 || varTrue; // true
// Donc :
console.log(result); // true

// Avec des parenthéses :
let result = varFalse && (varFalse || varTrue);
// Décomposons, parenthèses en premier :
let subResult1 = varFalse || varTrue; // true
let subResult2 = varFalse && subResult1; // false
// Donc :
console.log(result); // false
