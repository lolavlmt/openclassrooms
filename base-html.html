Les blocs HTML5

Les principales balises html pour organiser le contenu


        <header>Un logo, le menu de navigation (<nav>), etc.</header>
        <main> <!-- Élément unique par page, il contient le contenu principal variant d’une page à l’autre. -->
          <h1>Le titre de la page</h1>
          <section>
            <h2>Le titre de la section</h2>
            <article>
              <h3>Le titre de l'article</h3>
              <p>Le résumé de l'article</p>
              <aside> <!-- Son contenu est un complément par rapport à ce qui l'entoure, qui n'est pas forcément en lien direct avec le contenu mais qui peut apporter des informations supplémentaires. -->
                <p>Aparté</p>
              </aside>
            </article>
          </section>
        </main>
        <footer><!-- Tout comme la balise <header>, elle peut être utilisée au sein d'une <section> ou d'un <article>  -->
          Pied de page
        </footer>

Utilisation de class et id

Les attributs class et id, permettent d'identifier, de sélectionner, précisément un élément, pour pouvoir lui ajouter du style en css ou le manipuler en js.

L'attribut class est à privilégier autant que possible.

L'attribut id peut être utilisé pour faire une ancre, ou au sein des formulaires.

Attention, on ne nomme pas les id et les classes avec des accents, des espaces et on ne commence pas par un chiffre. On peut jouer sur la casse, les traits d’union ou les tirets bas (exemple : myBlock, my-block, my_bloc).

Par convention, on nomme les id et les classes en anglais.

Les ID

ID est un identifiant unique. Il ne peut être attribué qu'une seul fois au sein d'une page.

Un élément ne peut avoir qu'un seul id.
